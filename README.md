# VIKI
VIKI is a deep learning application for prediction stock price in KOSPI and KOSDAQ. 

# init 
```
virtualenv -p /usr/bin/python2.7 venv
. venv/bin/activate
pip install -r requirements.txt
```
# Before you commit 
```
pip freeze > requirements.txt
```
